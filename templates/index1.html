Simple Hashing Algorithm

Hashing is a fundamental concept used to convert data of any
size into a fixed-size string of characters, typically for indexing,
security, or data integrity purposes. It's like a fingerprint for data,
uniquely identifying it with a hash code.

  • This is a simple hashing algorithm which provides a random hash value for
    a given input.

    • Features:

  • It takes data in string format and provides a hash for the input
  • It can also read raw input from files.
